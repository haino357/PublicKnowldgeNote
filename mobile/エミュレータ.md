# Android
## キーボードの表示
エミュレータ右側ツールバーのキーボードアイコンをクリックして表示する。
表示されない場合は、`… (Extended controls) > Settings` で `Enable keyboard input` をオフにする。

## QRコード読み取り（Android エミュレータ）

### 前提条件
- Android Emulator 33.1.24 以降（Android Studio の SDK Manager で更新可能）
- エミュレータのカメラが **VirtualScene** に設定されていること

### 手順

#### 1. カメラ設定の確認
Android Studio の **Device Manager** でエミュレータの設定を開き、以下を確認する。

- **Show Advanced Settings** をクリック
- **Camera** セクションで **Back** を `VirtualScene` に設定

#### 2. QRコード画像を仮想シーンに表示する
1. エミュレータを起動する
2. 右側ツールバーの `… (Extended controls)` をクリック
3. **Camera** タブを選択
4. **Add image** をクリックし、読み取りたいQRコードの画像ファイルを選択
5. 仮想空間の壁面にQRコード画像が配置される

#### 3. カメラをQRコードに向ける
1. アプリ内でカメラ（QRコードリーダー）を起動する
2. エミュレータの仮想シーン内で、`Option (Alt)` キーを押しながらマウスを動かしてカメラの向きを操作する
3. QRコード画像が映るようにカメラを向けると自動的に読み取られる

### うまくいかない場合
- QRコード画像の解像度が低いと読み取れないことがある。500×500px 以上を推奨
- カメラの距離が近すぎる・遠すぎる場合は、`W/A/S/D` キーで仮想空間内を移動して調整する
- **Back Camera** が `Emulated` になっている場合は `VirtualScene` に変更して再起動する
