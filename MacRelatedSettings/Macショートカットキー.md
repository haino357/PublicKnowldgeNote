# Macショートカットキー

## ウインドウ位置の調整

`macOS Sequoia 15以降` の標準ショートカット。
`Fn` はキーボードによっては `Globe` キー。

### アクティブウインドウを移動・リサイズ

| 動作 | ショートカット |
| --- | --- |
| デスクトップいっぱいに広げる | `Fn` + `Control` + `F` |
| 中央に配置 | `Fn` + `Control` + `C` |
| 左半分に配置 | `Fn` + `Control` + `←` |
| 右半分に配置 | `Fn` + `Control` + `→` |
| 上半分に配置 | `Fn` + `Control` + `↑` |
| 下半分に配置 | `Fn` + `Control` + `↓` |
| 直前のサイズに戻す | `Fn` + `Control` + `R` |

### 複数ウインドウを一気に配置

| 動作 | ショートカット |
| --- | --- |
| 左右2分割 | `Fn` + `Control` + `Shift` + `←` |
| 右左2分割 | `Fn` + `Control` + `Shift` + `→` |
| 上下2分割 | `Fn` + `Control` + `Shift` + `↑` |
| 下上2分割 | `Fn` + `Control` + `Shift` + `↓` |
| 左1枚 + 右2枚 | `Fn` + `Control` + `Option` + `Shift` + `←` |
| 右1枚 + 左2枚 | `Fn` + `Control` + `Option` + `Shift` + `→` |
| 上1枚 + 下2枚 | `Fn` + `Control` + `Option` + `Shift` + `↑` |
| 下1枚 + 上2枚 | `Fn` + `Control` + `Option` + `Shift` + `↓` |

### 補足

- 四隅への単独配置は標準ショートカットなし
- Split View の左右配置やフルスクリーン化も一覧上はあるが、標準のキーボードショートカットは割り当てなし
